# 测试05：车内场景

## 难度：⭐⭐⭐⭐⭐

## 需求描述

创建简化的车内驾驶舱场景，重点展示氛围灯效果。

## 详细要求

### 场景设置
- 画布尺寸：1200 x 600 像素（21:9 比例）
- 背景色：黑色 (#000000)
- 相机位置：驾驶员视角 (0, 1.2, 0.5)，看向 (0, 1, -2)

### 车内结构（简化版）

```
俯视图：
    ┌─────────────────────┐
    │    ══════════       │  仪表台
    │   ┌───────┐         │  中控屏
    │   └───────┘         │
    │  ○           ○      │  方向盘（圆环）
    │ ┌──┐       ┌──┐     │  座椅
    │ └──┘       └──┘     │
    └─────────────────────┘
```

### 部件清单

| 部件 | 几何体 | 说明 |
|------|--------|------|
| 车厢外壳 | Box | 简化为长方体内部 |
| 仪表台 | Box | 前方横向长条 |
| 中控屏 | Plane | 发光平面 |
| 方向盘 | Torus | 圆环 |
| 座椅 x2 | Box | 简化方块 |

### 氛围灯（重点）

| 位置 | 类型 | 颜色 | 说明 |
|------|------|------|------|
| 仪表台下沿 | 灯带 | 可变 | 主氛围灯 |
| 车门边缘 | 灯带 | 可变 | 左右各一条 |
| 中控屏背光 | 面光源 | 蓝白 | 屏幕发光 |

### 灯带实现方式
- 使用 TubeGeometry 或多个小 Box 排列
- 材质：MeshBasicMaterial（自发光，不受光照影响）
- 支持颜色动态变化（通过参数控制）

### 交互功能
- OrbitControls（限制角度，模拟车内视角）
- 提供颜色切换按钮（红/蓝/绿/紫）
- 氛围灯呼吸效果（亮度渐变）

### 输出格式
- 单个 HTML 文件

## 验收标准

- [ ] 车内基本结构可辨认
- [ ] 氛围灯正确发光
- [ ] 颜色可切换
- [ ] 呼吸效果流畅
