# 测试06：简化汽车外观

## 难度：⭐⭐⭐⭐

## 需求描述

使用基础几何体组合创建简化的汽车外观模型。

## 详细要求

### 场景设置
- 画布尺寸：800 x 600 像素
- 背景色：浅灰 (#e0e0e0)
- 相机位置：(5, 3, 5)，看向原点

### 汽车结构（简化版）

```
侧视图：
       ┌───────┐
      /         \        车顶（梯形/弧形）
    ┌─────────────┐
    │             │      车身主体
    │  ○       ○  │      车轮
    └─────────────┘

俯视图：
    ┌─────────────┐
    │             │
    │             │
    └─────────────┘
```

### 部件清单

| 部件 | 几何体 | 尺寸 | 颜色 |
|------|--------|------|------|
| 车身 | Box | 4 x 1 x 2 | #2c3e50 |
| 车顶 | Box | 2 x 0.8 x 1.8 | #2c3e50 |
| 车轮 x4 | Cylinder | r=0.4, h=0.3 | #1a1a1a |
| 车窗 | Box | 1.8 x 0.6 x 0.05 | #87ceeb |
| 车灯 x2 | Sphere | r=0.15 | #ffff00 |

### 材质
- 车身：MeshStandardMaterial，金属度 0.6
- 车窗：MeshBasicMaterial，半透明
- 车轮：MeshStandardMaterial，粗糙度 0.9

### 灯光
- 环境光：强度 0.4
- 方向光：位置 (5, 10, 5)，投射阴影

### 地面
- 平面 20 x 20
- 颜色：#cccccc
- 接收阴影

### 交互
- OrbitControls 旋转查看
- 汽车缓慢旋转展示

### 输出格式
- 单个 HTML 文件

## 验收标准

- [ ] 汽车外形可辨认
- [ ] 各部件比例协调
- [ ] 阴影效果正常
- [ ] 可旋转查看
