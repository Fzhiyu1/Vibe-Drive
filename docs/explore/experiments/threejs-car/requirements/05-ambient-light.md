# 测试05：氛围灯带场景

## 难度：⭐⭐⭐

## 需求描述

创建一个展示氛围灯带效果的场景，专注于灯光表现而非复杂建模。

## 详细要求

### 场景设置
- 画布尺寸：800 x 600 像素
- 背景色：纯黑 (#000000)
- 相机位置：(0, 2, 5)

### 场景结构

```
俯视图：
    ┌─────────────────┐
    │  ═══════════    │  顶部灯带
    │                 │
    │ ║             ║ │  左右灯带
    │ ║             ║ │
    │                 │
    │  ═══════════    │  底部灯带
    └─────────────────┘
```

### 灯带参数

| 位置 | 长度 | 宽度 | 位置坐标 |
|------|------|------|----------|
| 顶部 | 4 | 0.1 | (0, 2, -2) |
| 底部 | 4 | 0.1 | (0, 0, -2) |
| 左侧 | 2 | 0.1 | (-2, 1, -2) |
| 右侧 | 2 | 0.1 | (2, 1, -2) |

### 灯光效果
- 材质：MeshBasicMaterial（自发光）
- 默认颜色：蓝色 (#00aaff)
- 发光强度可调

### 动画效果
- **呼吸效果**：亮度在 0.3-1.0 之间渐变，周期 2 秒
- **流光效果**：颜色沿灯带移动（可选）

### 交互
- 颜色选择器（红/蓝/绿/紫/白）
- 开关呼吸效果

### 输出格式
- 单个 HTML 文件

## 验收标准

- [ ] 四条灯带正确显示
- [ ] 自发光效果明显
- [ ] 呼吸动画流畅
- [ ] 颜色可切换
