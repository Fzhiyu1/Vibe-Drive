# 测试01：球形

## 难度：⭐

## 需求描述

创建一个可交互的 3D 球体场景。

## 详细要求

### 场景设置
- 画布尺寸：800 x 600 像素
- 背景色：深灰色 (#1a1a1a)
- 相机位置：z = 5

### 球体参数
- 半径：1
- 分段数：32 x 32
- 颜色：蓝色 (#4a90d9)
- 材质：MeshStandardMaterial（有光泽感）

### 灯光
- 环境光：白色，强度 0.4
- 点光源：白色，强度 1.0，位置 (5, 5, 5)

### 交互
- 鼠标拖拽旋转视角（OrbitControls）
- 球体自动缓慢旋转（Y轴，速度 0.01 rad/frame）

### 输出格式
- 单个 HTML 文件，内联 JavaScript
- 使用 CDN 引入 Three.js

## 验收标准

- [ ] 球体正确显示
- [ ] 光照效果正常
- [ ] 可拖拽旋转
- [ ] 球体自动旋转
