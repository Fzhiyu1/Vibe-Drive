# 测试03：小人

## 难度：⭐⭐

## 需求描述

使用基础几何体组合创建一个简易火柴人/机器人小人。

## 详细要求

### 场景设置
- 画布尺寸：800 x 600 像素
- 背景色：深蓝色 (#0d1b2a)
- 相机位置：(0, 2, 5)，看向 (0, 1, 0)

### 小人结构（使用基础几何体组合）

```
      ○        头部：球体，半径 0.25
      |
    ──┼──      身体：圆柱体，高 0.8，半径 0.15
      |        手臂：圆柱体 x2，高 0.5，半径 0.05
     / \       腿部：圆柱体 x2，高 0.6，半径 0.08
```

| 部位 | 几何体 | 尺寸 | 位置 Y | 颜色 |
|------|--------|------|--------|------|
| 头 | 球体 | r=0.25 | 1.7 | #f5d0c5 |
| 身体 | 圆柱 | h=0.8, r=0.15 | 1.1 | #3498db |
| 左臂 | 圆柱 | h=0.5, r=0.05 | 1.2 | #f5d0c5 |
| 右臂 | 圆柱 | h=0.5, r=0.05 | 1.2 | #f5d0c5 |
| 左腿 | 圆柱 | h=0.6, r=0.08 | 0.3 | #2c3e50 |
| 右腿 | 圆柱 | h=0.6, r=0.08 | 0.3 | #2c3e50 |

### 灯光
- 环境光：强度 0.5
- 点光源：位置 (3, 5, 3)，强度 1.0

### 交互
- OrbitControls
- 小人整体缓慢旋转展示

### 输出格式
- 单个 HTML 文件

## 验收标准

- [ ] 小人各部位正确组合
- [ ] 比例协调
- [ ] 可旋转查看
