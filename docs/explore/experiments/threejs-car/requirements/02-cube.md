# 测试02：方形

## 难度：⭐

## 需求描述

创建一个带纹理的 3D 立方体场景。

## 详细要求

### 场景设置
- 画布尺寸：800 x 600 像素
- 背景色：深灰色 (#1a1a1a)
- 相机位置：(3, 3, 3)，看向原点

### 立方体参数
- 尺寸：1 x 1 x 1
- 材质：MeshStandardMaterial
- 颜色：橙色 (#e67e22)
- 金属度：0.3
- 粗糙度：0.4

### 灯光
- 环境光：白色，强度 0.3
- 方向光：白色，强度 0.8，位置 (5, 10, 7)
- 方向光投射阴影

### 地面
- 平面几何体：10 x 10
- 颜色：灰色 (#333333)
- 接收阴影

### 交互
- OrbitControls 旋转视角
- 立方体悬浮在地面上方 0.5 单位

### 输出格式
- 单个 HTML 文件，内联 JavaScript
- 使用 CDN 引入 Three.js

## 验收标准

- [ ] 立方体正确显示
- [ ] 阴影效果正常
- [ ] 地面可见
- [ ] 可拖拽旋转视角
