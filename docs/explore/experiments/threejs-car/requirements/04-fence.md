# 测试04：栅栏

## 难度：⭐⭐

## 需求描述

创建一段木质栅栏，测试重复几何体和材质表现。

## 详细要求

### 场景设置
- 画布尺寸：800 x 600 像素
- 背景色：天空蓝渐变 (#87ceeb)
- 相机位置：(0, 2, 6)

### 栅栏结构

```
  |   |   |   |   |   |      竖板：5 根
  |   |   |   |   |   |
=======================      横梁：2 根
  |   |   |   |   |   |
=======================
  |   |   |   |   |   |
```

| 部件 | 数量 | 尺寸 (宽x高x深) | 间距 |
|------|------|-----------------|------|
| 竖板 | 5 | 0.15 x 2.0 x 0.1 | 0.8 |
| 横梁 | 2 | 4.0 x 0.15 x 0.1 | Y=0.5, Y=1.5 |

### 材质
- 颜色：木色 (#8b4513)
- 粗糙度：0.8
- 金属度：0

### 地面
- 草地颜色：#228b22
- 尺寸：10 x 10

### 灯光
- 环境光：暖白色，强度 0.6
- 方向光：模拟阳光，位置 (5, 10, 5)

### 输出格式
- 单个 HTML 文件

## 验收标准

- [ ] 栅栏结构正确
- [ ] 木质感材质
- [ ] 草地背景
