# 探索文档

## 什么是探索文档

探索文档是一种**话题驱动的技术调研文档**，用于：

1. **讨论** - 围绕特定话题进行结构化讨论
2. **验证** - 通过实验代码验证讨论结论
3. **对比** - 测试不同方案或不同 AI 的生成能力

## 文档结构

```
explore/
├── README.md                    # 本文件
├── 探索文档-[主题].md            # 讨论记录
└── experiments/                 # 实验代码
    └── [实验名]/
        ├── README.md            # 实验说明
        ├── requirements/        # 需求文档
        ├── claude/              # Claude 生成
        ├── gemini/              # Gemini 生成
        └── baseline/            # 基准代码
```

## 工作流程

1. 创建探索文档，列出待讨论话题
2. 逐个话题讨论，记录结论
3. 需要验证时，创建实验需求
4. 用不同方案/AI 生成代码
5. 对比结果，更新结论
