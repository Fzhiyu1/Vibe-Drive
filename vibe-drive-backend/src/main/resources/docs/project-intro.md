# Vibe Drive 项目介绍

## 项目概述

Vibe Drive 是一个基于 AI Agent 的车载智能氛围编排系统，通过环境感知驱动"时空叙事"体验。

## 核心功能

### 1. 环境感知
- GPS 位置识别（高速、隧道、城市等）
- 天气状况感知
- 时段识别
- 驾驶员状态监测

### 2. 氛围编排
- **音乐推荐**：根据场景智能推荐音乐
- **灯光控制**：动态调节车内氛围灯
- **香氛控制**：智能释放香氛
- **按摩座椅**：根据疲劳度调节按摩

### 3. 安全模式
- L1 正常模式：全功能
- L2 专注模式：减少干扰
- L3 静默模式：高速行驶时最小化交互

## 技术架构

- **后端**：Spring Boot 3 + LangChain4j
- **前端**：Vue 3 + TypeScript + Three.js
- **AI**：DeepSeek API
- **语音**：Whisper (STT) + Edge TTS

## 演示说明

这是一个演示系统，不连接真实车机硬件。所有环境数据均为模拟数据。
